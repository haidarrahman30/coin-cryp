# CryptoCoin App

Aplikasi ini ialah aplikasi latihan frontend yang saya gunakan untuk membangun aplikasi yang memberikan informasi seputar koin crypto dan pergerakannya.

## Implementasi

-  React JS
-  Reusable Component
-  Dark Mode Light Mode 
-  Eslint
-  Responsive Design

## Cara menjalankannya
1. npm install
2. npm start
